# FABRICK Admin Panel - Brand Guidelines

## Vizuálny štýl (Tailwind CSS)
- **Hlavná farba (Branding):** Vínová / Burgundy (`bg-[#5f2524]`, Pantone 491 C).
- **Doplnková farba:** Čistá biela (`bg-white`) pre karty a panely, tmavošedá (`text-neutral-800`) pre čitateľné texty.
- **Dizajn systém:** Čisté, moderné B2B rozhranie. Zaoblenia rohov `rounded-xl`, jemné tiene `shadow-sm`, vzdušný layout s dostatočným paddingom (`p-6`).
- **Typografia:** Bezpätkové moderné písmo (Inter alebo systémové sans-serif), tučné nadpisy (`font-bold`).
